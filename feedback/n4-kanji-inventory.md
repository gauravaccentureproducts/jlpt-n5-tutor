## Native-teacher review notes

- **Borderline placements** (entries that some sources treat as N5 prerequisite): 会, 事, 自, 社, 言, 兄, 弟, 妹, 姉, 父, 母 — these all appear in N5 vocab compounds but the kanji-recognition test typically debuts in N4. The N4 build should consult Tanos for the more conservative classification.
- **Common-mistake reading attribution risks** (per Pass-13 lessons): each kanji's PRIMARY on-yomi and kun-yomi must be verified against Tanos; JLPT Sensei occasionally presents non-frequency-ordered readings.
- **JA-13 invariant** (no out-of-scope kanji): when this list is committed as `data/n4_kanji_whitelist.json`, the N4 build's integrity check uses this list as the scope reference. The whitelist should be the UNION of N5 + N4 kanji per §B.10 (the procedure manual's recommended composition rule); each entry carries `tier: prerequisite_n5 | core_n4 | late_n4` accordingly.
