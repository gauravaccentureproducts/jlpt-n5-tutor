# JEES inquiry template

> **Status:** ready to send when needed. Not currently sent.
>
> The corpus is original (`CONTENT-LICENSE.md`); we don't presently
> need a past-paper license for the project to function. This template
> exists so that *if* the project ever wants to add a "compare your
> score against actual past papers" feature — or *if* JEES contacts us
> first with a question — there's a starting point that doesn't need
> to be drafted under time pressure.

## When to use this

Send this email if:

- You want to use **specific past-paper questions** (verbatim text)
  in the project, with proper attribution.
- You want to publish **translations** of past-paper questions for
  non-Japanese-speaking learners.
- You want to add a **comparison feature** that scores users against
  the actual JLPT past-paper distribution (would require licensed
  past-paper data, not just our originals).
- JEES has contacted you with a concern and you need to respond
  formally.

Do **NOT** send this if you only want to:
- Continue authoring original questions in JLPT format → no
  permission needed; already covered by `CONTENT-LICENSE.md`
- Reference the JLPT format / Mondai structure / question count
  distribution → that's factual information, not copyrighted
- Cite jlpt.jp's free resources for learners → fair quotation
  (引用) under Japanese Copyright Act Article 32

## What to expect

JEES typically responds within 1–4 weeks via email or postal mail.
Possible outcomes:

| Response | Likelihood | What to do |
|---|---|---|
| Granted (with conditions: attribution, audience cap, fee) | Medium for non-commercial educational projects | Comply with conditions; add JEES attribution to each licensed question |
| Granted with a referral to a third-party licensor (e.g., 凡人社, the official past-paper book publisher) | Medium | Follow up with the licensor JEES names |
| Denied with no further explanation | Low–medium | Continue with original content; do not republish past papers |
| No response | Low for direct email; medium for the contact form | Wait 4 weeks, then follow up once via the alternate contact (postal if you emailed first, or vice versa) |
