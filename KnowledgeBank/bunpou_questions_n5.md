# JLPT N5 Bunpou (文法) Practice Questions

100 questions covering the Bunpou (grammar) section of the JLPT N5. The Bunpou section tests three skills: choosing the correct grammar form, arranging words into a sentence, and using grammar across a connected passage.

## Subtypes covered

| Mondai | Subtype | Count |
|---|---|---|
| Mondai 1 | 文の文法1 (sentence grammar 1) - choose the correct grammar form | 60 |
| Mondai 2 | 文の文法2 (sentence composition / 並べ替え) - arrange 4 elements, identify what goes in the starred position | 30 |
| Mondai 3 | 文章の文法 (text grammar) - short passage with multiple grammar blanks | 10 |

## Notation rules

- The blank in Mondai 1 is shown as `（  　）`.
- In Mondai 2, the four shuffled elements are listed as 1-4 and the original sentence shows positions A B C D where one is marked `★`. The answer identifies which option-number goes in the `★` slot.
- Mondai 3 passages have multiple inline blanks numbered `[ 1 ]` `[ 2 ]` etc., each with its own 4-option set.
- All stems and correct answers use only N5-syllabus kanji. Distractors may include non-N5 kanji where authentic JLPT format requires it.
- No em dashes (U+2014) appear in this file.

## Engine display note

For mock-test mode, the app's test engine MUST hide the `**Answer:**` line and rationale until the student commits an answer. The visible-by-default format here is for self-study reference; runtime test rendering is the engine's responsibility.

## Numeral convention

Numbers are written using both kanji forms (一, 二, 三, 五, 十, 百, 千) and arabic numerals (1, 2, 100, 1000) - mirroring authentic JLPT papers, which use kanji numerals in narrative text and arabic numerals in prices, addresses, schedules, and time tables. This is intentional, not inconsistency.

## Mondai 3 - 文章の文法 (Text Grammar)

10 questions across 2 short passages. Each passage has 5 numbered blanks; pick the best fit for each.

### Passage A (Q91-Q95)

> たなかさんの 一日

> わたしは まいにち あさ 六時 [ 1 ] おきます。 七時ごろ [ 2 ] あさごはんを 食べます。 あさごはんは いつも パン [ 3 ] ぎゅうにゅうです。 ８時に いえを 出て、電車で 学校 [ 4 ] 行きます。 学校は 九時 [ 5 ] 三時までです。

#### Q91 (blank 1)

1. に
2. を
3. が
4. で

**Answer: 1** - time + に.

#### Q92 (blank 2)

1. に
2. が
3. を
4. へ

**Answer: 1** - approximate time (七時ごろ) + に for time of action.

#### Q93 (blank 3)

1. に
2. と
3. を
4. の

**Answer: 2** - listing と.

#### Q94 (blank 4)

1. で
2. を
3. へ
4. が

**Answer: 3** - direction. (Note: に would also be grammatically valid for a motion-verb destination at N5, so it has been replaced with で as a clearly-wrong distractor: 学校で 行きます would mark 学校 as the location of the going-action, which is ungrammatical.)

#### Q95 (blank 5)

1. へ
2. を
3. から
4. に

**Answer: 3** - 9 o'clock from / starting at.

### Passage B (Q96-Q100)

> わたしの しゅみ

> わたしは おんがく [ 1 ] すきです。 まいにち、 学校から かえってから、いえで ピアノを 一時間 [ 2 ]。 にちようびは 友だちと いっしょに [ 3 ] に 行きます。 友だち も ピアノが じょうずですから、 とても たのしいです。 こんしゅうは いそがしくて、ピアノを ひく 時間 [ 4 ] ありませんでした。 来週は ぜったいに 一日 [ 5 ] ひきたいと おもいます。

#### Q96 (blank 1)

1. に
2. が
3. を
4. と

**Answer: 2** - すき takes が.

#### Q97 (blank 2)

1. ひきます
2. ひいて います
3. ひきました
4. ひきたいです

**Answer: 1** - habitual present.

#### Q98 (blank 3)

1. ピアノを ひいて
2. ピアノきょうしつ
3. ピアノで うた
4. ピアノは すき

**Answer: 2** - 「ピアノきょうしつ」 (compound noun, no の) - に行きます target. (The original option 4 「ピアノを 買い」 was dropped because Verb-stem + に+行く is grammatically valid, making it a structurally-correct second answer; replaced with 「ピアノは すき」 which is broken before に行きます.)

#### Q99 (blank 4)

1. が
2. を
3. に
4. の

**Answer: 1** - time + が + ありません (no time).

#### Q100 (blank 5)

1. ぐらい
2. しか
3. だけ
4. でも

**Answer: 4** - 「でも」 attached to a quantity means "even (just)" - 一日でも = "even (just) one day". Combined with 「ぜったいに」, this expresses "I definitely want to play, even just for one day next week." 「ぐらい」 (approximate) would clash with 「ぜったいに」 (firm resolve).
